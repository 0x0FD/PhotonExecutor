# Photon Executor ⚡

**Photon Executor** is an open-source script executor powered by **Xeno**, designed to be fast, lightweight, and simple to use.

🚀 **Fast**  
⚙️ **Native Xeno backend**
---

## ✨ Features

- Monaco-based code editor (WebView2)
- One-click attach using Xeno
- Execute scripts on attached clients
- Modern dark UI
- Lightweight & fast startup
- Open-source and customizable

---

## 📦 Requirements

- Windows 10 / 11 (x64)
- .NET 6.0+ Runtime
- Microsoft Edge WebView2 Runtime
